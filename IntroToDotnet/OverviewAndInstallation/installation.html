Install

For this course, we need to first get a few things installed before rolling right into the material. Some of these things might not be used until a little later, but it is nice to get them out of the way upfront.

Installing Visual Studio Code

While it is entirely possible to go through this course using any text editor or a full IDE like full Visual Studio, we highly recommend installing and working with Visual Studio Code. It is an excellent tool that provides more than standard text editors, but is more lightweight than a full IDE like Visual Studio.

All you need to do to get the latest version of VS Code is go to https://code.visualstudio.com and click the download button.

Installing the 'C# for Visual Studio Code' plugin from OmniSharp

The OmniSharp plugin will give us a ton of extra features when working with C# code in Visual Studio Code, such as ASP.NET syntax highlighting, early (pre-compile) error detection, code suggestions and completion, and insight into different class functionality. It is easy to add:

  1. Click on the Extensions icon
  2. Search for "OmniSharp" or "C#" (the one you want to install will actually be called "C#")
  3. Click Install
  4. Once installed, click Reload

Installing .NET Core

Even though .NET Core is relatively young, there have already been several versions released. We are currently using SDK Version: 2.0.3, which you can download here by first selecting the correct version, and then selecting the installer specific to your operating system.

If you had any terminal windows open during installation (including in VS Code), be sure to restart them before proceeding so that the dotnet commands will be recognized in your terminal.

MacOS Specific

Before installing the .NET Core SDK on Mac OS, you need to have the latest version of OpenSSL set up. The easiest way to get OpenSSL is by using Homebrew. If you do not already have Homebrew, you can download and install it by going to the Homebrew webpage. Once installed, do the following in terminal: 

brew install openssl
ln -s /usr/local/opt/openssl/lib/libcrypto.1.0.0.dylib /usr/local/lib/
ln -s /usr/local/opt/openssl/lib/libssl.1.0.0.dylib /usr/local/lib/

From there, we can download the .NET Core SDK for Mac by clicking the link above.

Installing MySQL Server

Note: if you have taken Python or PHP, you most likely already have MySQL installed and can skip this step. Otherwise you may review the installation steps from the Python SQL Chapter. Test your MySQL server connection by invoking a MySQL shell from your terminal: (the following command assumes you have an administrative user "root") 

mysql -u root -p

After you type in your password, the prompt should change to:

mysql>

which lets you know you are now logged into your MySQL server. To quit, just type quit or exit.

NuGet

Microsoft's family of .NET products uses a package manager called NuGet. A package manager is simply a tool that helps us bring outside packages into our code. The core NuGet features we need actually came with our SDK installation, so we don't need to do any additional work to use it, but we will be using it indirectly throughout the course to install various packages. (Most package managers are designed for use with a specific language or framework. In Python we used pip, node has npm, Ruby has RubyGems).

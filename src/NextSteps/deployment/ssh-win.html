Connecting (For PC)

Let's access this computer. We can access our files on our local machine in two main ways. We can either use a GUI like Windows Explorer to see our files or we can use the command prompt to access the file structure of our computer. To access our EC2 instance, we are going to use a program called PuTTy. This is really cool. It's as if we are right next to the EC2 instance that we bought, and we are running the PuTTy application on that computer. This also means that we won't have access to a GUI like Windows Explorer to access the file structure, so it is crucial that we know general terminal commands to navigate the files.

Generate a .ppk file

  1. Using your PC, Download WinSCP. You can download it here. Using WinSCP, we can then generate a .ppk file ( This is an extra step for a Windows OS because we can't use Command Prompt to access our Ubuntu computer. We use PuTTy to communicate to our server and in order for us to do that, we need to convert .pem file to .ppk file.
  2. Open WinSCP and click on Tools > Run PuttyGen
  3. Click on Load and locate your .pem "myfirstinstance" key.


  4. Key in your passphrase under "Key passphrase and Confirm Passphrase" and hit Save private key. Make sure you take note of your passphrase, you will be using this to log in to your Ubuntu computer.

  5. Name your Putty Private Key .ppk to myfirstinstance and hit Save


  6. Close Your PuTTy Key Generator and WinSCP

Log in to our Ubuntu Server

  1. Run Putty and paste in your Instance's Public IP under Host Name. Port should be Port 22 (Remember the default port number when we first configured our Ubuntu Computer? Yes SSH Port 22!):


  2. Still, on the PuTTy window, expand SSH and click Auth and browse for the "myfirstinstance.ppk" file.
  3. If you don't want to keep doing this step every time you want to log in to your Ubuntu computer, you can scroll up, click Session and save this session as " myfirstinstance", then click Save.

  4. Now click Open to open the PuTTy terminal. Once in the PuTTy console, you will see a text that says "login as:" type in ubuntu as user name (the default username). Then type your "passphrase" we saved earlier (In Generate a .ppk file - Step 4 above).

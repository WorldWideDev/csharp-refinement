Forum

Create a forum website with basic users, moderators, and administrators.

Basic users should be able to:
  • Create topics
  • Respond to topics
  • Edit their profile information
  • Delete their own posts (but not topics)
Moderators should be able to:
  • Do everything basic users can
  • Each moderator is associated with one or more topic categories
  • Delete topics within their category(ies)
  • Delete basic users' posts in their category(ies)
Administrators should be able to:
  • Do everything moderators can, across all topic categories
  • Delete any user other than another administrator
  • Edit any users' information other than another administrator
  • Promote other users to Moderator of any topic category
  • Create topic categories
  • Delete topic categories

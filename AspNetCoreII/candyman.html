The Candyman Can!

Up to this point, we've been writing all of our projects from scratch and after awhile we start to notice there always seems to be a lot of base code we have to have. This is what we call boilerplate and it would be nice if we could just automate this boilerplate code generation. Well, there's a way to do this! Programmers are lazy and don't want to do the task if they can increase overall productivity skipping them. As such, this generator called scaffolding exist for us to use. In this case, the one we will be using is called Yeoman and with Yeoman, we will use the CANDYMAN package to build out a solid foundation of an ASP.NET Core project for us from here on out.

Before installing Yeoman, we will need to make sure we already have the following:

  • Node.js v4 or higher
  • npm
  • git

If you do not have these, you can get the latest version of Node here (Note that npm comes packaged with Node). You should have git installed on your machine at this point, but just in case it can be found here.

Installing Yeoman and CANDYMAN

With Node and npm installing Yeoman is simple and straight forward! Just open a terminal and follow the steps below. 

sudo npm -g update
npm install -g bower
npm install -g yo

Copy

Note if you are using the older .NET Core version (Pre-1.1 Release) use the following commmand

npm install -g generator-candyman@0.1.95

Copy

Otherwise simply use:

npm install -g generator-candyman

Copy

If you have no errors during installation you are good to go and can check your install by typing 

yo --version

Using Candyman

The generator can be used in a couple of ways. Typing the yo candyman command will give you a prompt to fill in all information needed or you can skip some steps by filling in the third optional parameter of the app name. 

yo candyman <your app name>

Copy

After the generator has finished running all you must do is navigate to the new directory that was created (the project name) and run dotnet restore

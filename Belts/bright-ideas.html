For a Red Belt:

  • Login and Registration with Validations
    • Validation messages should display for unsuccessful submissions
    • (Registration) Name should be only letter characters and spaces
    • (Registration) Alias should be only letter and numbers characters
    • (Registration) Email should be valid Email address
    • (Registration) Password with a minimum length of 8 characters
    • (Login) Valid Email and Password
  • Dashboard
    • Logged in user can delete thier own ideas
    • New Idea must be at least 5 characters long
    • Each Idea displays the correct number of "likes" it has received
    • Clicking "like" on Idea add +1 to the number of likes
  • Like Details
    • Display the Alias' of Users that have like the Idea
    • Display Alias of User that created the Idea, as well as the content of the Idea
  • User Details
    • Display User's Name, Alias, and Email

For a Black Belt:

  • (Mandatory) Exam is deployed to either Amazon EC2 or Microsoft Azure. IP address to hosted site should be included in a text file.
  • (Mandatory) All Red Belt features.
  • 2/3 of the following features:
    • (Dashboard) Ideas are arranged by total number of likes received, descending.
    • (Dashboard) A user can only like an Idea once.
    • (User Detials) Display the total number of Ideas created by the User, plus the number of Likes all of their Ideas have received
